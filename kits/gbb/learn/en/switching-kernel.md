# Switching Kernel

[Prev]() [Next]()

When creating a project, if you choose not to use a starter kit, you can select an installed kernel. (Or if you select a starter kit, it will inherit its kernel choice.)

<img src="imgs/editor-project-creating-selecting-kernel.png" class="diagram-image diagram-screenshot">

<div class="small-note">Creating project with a kernel</div>

You can switch the kernel used by a project at any time in the project property dialog.
