# Releasing

[Prev]() [Next]()

This section provides an overview of releasing games built with GB BASIC. The following subsections will explain how to export a ROM or package your project for web and desktop distribution.

For ROMs and standalone executables you have the copyright to, you are free to distribute them digitally or as physical cartridges.
