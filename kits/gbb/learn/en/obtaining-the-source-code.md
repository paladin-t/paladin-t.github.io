# Obtaining the Source Code

[Prev]() [Next]()

Two recommended ways to obtain the kernel source code:

1. Eject the VM source code directly from the Installed Kernels dialog or the Help menu within a desktop environment
2. Visit <a class="nav-link" href="https://github.com/paladin-t/gbb/releases" target="_blank">GB BASIC Releases - GitHub <i class="fa-solid fa-up-right-from-square"></i></a> and download the source code for the relevant release version

<img src="imgs/kernel-development-eject-vm-from-installed-kernel.png" class="diagram-image diagram-screenshot">

<div class="small-note">Eject source code from the Installed Kernels dialog</div>

<img src="imgs/kernel-development-eject-vm-from-recent-screen.png" class="diagram-image diagram-screenshot">

<div class="small-note">Eject source code from the Help menu</div>

Using the two recommended methods above, you can obtain the latest stable version of the source code. In addition, visiting <a class="nav-link" href="https://github.com/paladin-t/gbb/tree/main/src/vm" target="_blank">gbb/src/vm - GitHub <i class="fa-solid fa-up-right-from-square"></i></a> will provide the latest working source code, which may include some unreleased features under development. Typically, you do not need to use this version of the source code for building custom kernels.
