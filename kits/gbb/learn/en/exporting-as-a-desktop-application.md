# Exporting as a Desktop Application

[Prev]() [Next]()

This feature is available in the desktop version with the Desktop Builder DLC.

Click "Build", then select a desktop platform target in the main menu to start the desktop export process.

<img src="imgs/building-for-desktop-entry.png" class="diagram-image diagram-screenshot">

<div class="small-note">Building for desktops</div>

The build will use the current project's property settings. You can adjust the options in the pop-up build settings dialog.
