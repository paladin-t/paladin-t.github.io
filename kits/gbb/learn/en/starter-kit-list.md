# Starter Kit List

[Prev]() [Next]()

The current latest version includes the following starter kit options.

* **None**: Blank project; you can choose a kernel by your own
* **Minimal**: A game framework that shows how to make a minimal runnable project; using the default kernel
* **Platformer**: A program that implements a platformer game; using the default kernel
* **Top-down**: A program that implements a top-down game; using the default kernel
* **Point and Click**: A program that constructs game scenes with simple `image` as background, manually creates player, objects and triggers, alongside callbacks; using the default kernel
* **Text Adventure**: A program that constructs game scenes with `image` as background, and `label` based text to show the story and interactable options, manually creates player and trigger alongside callbacks; using the default kernel
* **Reversi**: A program that constructs a Reversi game with its rule, human and CPU players; using the default kernel
* **Music Player**: A program that implements a simple music player and includes several music; using the default kernel
* **Scroll Shooting**: A program that implements a scroll shooting game; using the scroll shooting kernel
* **Mandelbrot Demo**: A program that computes and plots the Mandelbrot set with graphics primitives; using the default kernel
